Blockman
============

Blockman is a chrome extension for for blocking unwanted web requests.

Building
========

For building, run:

```
   $ bash build.sh
```

For building in development mode, run:

```
   $ bash build.sh develop
```

This will result in `dist/Blockman-develop` directory.
